INSTALLAZIONE STEP BY STEP DI TouRES SErvice SU WINDOWS

PASSO1: installare la versione 3.6.9 di Python recandosi sul sito https://www.python.org/downloads/ (non installare la versione 3.7, potrebbe creare problemi con l'installazione di tensorflow
		ancora in fase beta per Python 3.7)

PASSO2: aprire il prompt dei comandi e creare un ambiente virtuale digitando il comando: pip install virtualenv

PASSO3: eseguire il comando: pip install numpy scipy keras tensorflow pandas flask flask-cors psutil sklearn
			
PASSO5: eseguire lo script "runNetworks.bat" (doppio clic sullo script), attendere la chiusura della CLI

PASSO6: eseguire lo script "activate_server.bat" (doppio clic sullo script)

PASSO7: eseguire lo script "updater_system.bat" (doppio clic sullo script)
